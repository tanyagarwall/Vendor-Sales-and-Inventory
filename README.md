# Vendor-Sales-and-Inventory
Built an end-to-end ETL pipeline using Python and SQLite, performed EDA, and created Power BI dashboards to analyze vendor performance, sales, and inventory efficiency

## Introduction
This project focuses on analyzing vendor performance, inventory management, and sales operations using data analytics techniques. Developed during my internship at
DRDO SAG Lab, Delhi, the project aims to transform raw procurement and sales data into meaningful business insights through data cleaning, ETL processes,
exploratory data analysis, statistical analysis, and interactive dashboard visualization. The system helps identify top-performing vendors, optimize inventory
turnover, analyze profitability, and support data-driven business decision-making.

## Technologies Used
1) Python
2) SQLite
3) Power BI
4) Pandas
5) Matplotlib

## Features
1) Vendor performance analysis                                                                                          
2) Inventory turnover evaluation
3) Sales and profitability analysis
4) ETL pipeline for data processing
5) Data cleaning and transformation
6) Exploratory Data Analysis (EDA)
7) Statistical analysis and KPI generation
8) Interactive Power BI dashboards
9) Cost and procurement analysis
10) Business insight generation through visualizations
